## Evening Session Additions (9PM ET)

### Scanner Build — Key Learnings
- **yfinance rate limiting is REAL**: rapid-fire calls return None silently. Solution: 1s delay per ticker minimum, OR use `news_universe.json` cached universe to avoid re-scraping tickers
- **Scanning individual tickers works, bulk loops fail**: Import-time globals + rapid API = corrupted data. Fix: standalone scripts, rate-limited
- **News ticker extraction is noisy**: Regex from news HTML = too much garbage. Better: curated universe + news filtering
- **Final scanner stack**: curated_universe.py (43 stocks) → news_scanner.py (gap/squeeze analysis) → run_news_scan.py → news_scan_results.json
- **News sources tried**: Finviz, Benzinga, StockTwits (blocked), Google News RSS, Reddit WSB (403), Seeking Alpha — filtered to curated universe as fallback
